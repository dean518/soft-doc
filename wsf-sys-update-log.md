
#### 介绍

wsf  是一个基于swoole 的 php   高性能分布式常驻后台基于TCP的RPC框架  
主要的作用是 用来进行多任务数据处理   和比较耗时的计算任务等 以及异步任务等
> 兼容环境
- Windows 7-10  (这里只支持部分功能 包括 通信 以及多进程)
- Cygwin   基本支持全功能
- Linux    全功能支持
